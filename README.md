### **Introduction**

------

通过数据库技术和自然语言处理相关技术智能开发质性资料，探索LLM在质性资料研究中的实际应用。借助智慧问答、文本生成、语义理解和信息提取等任务，开展AI＋媒体，AI＋新闻的跨学科研究，为深度访谈提供大模型时代的研究思路。结合情感分析技术，构建详尽的被访者人物画像，提供全面的研究视角。
